HEX vs RGB in HTML

In HTML/CSS, colors can be defined using HEX or RGB values.
HEX (hexadecimal): Starts with # followed by 6 characters.
Example: #FF0000 is red.
Format: #RRGGBB (Red, Green, Blue in hex: 00–FF)
RGB: Uses the rgb() function with 3 numbers from 0–255.
Example: rgb(255, 0, 0) is also red.
Format: rgb(red, green, blue)
Both define the same colors, just different formats. Use whichever you prefer!
